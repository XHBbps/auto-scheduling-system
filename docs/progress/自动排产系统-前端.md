## 项目架构

- Vue 3 / TypeScript / Vite / Element Plus
- 视图层：`apps/web/src/views`
- 页面编排与业务状态：`apps/web/src/composables`
- 展示组件：`apps/web/src/components`
- 路由：`apps/web/src/router`
- 工具与类型：`apps/web/src/utils`、`apps/web/src/types`
- 构建与观测脚本：`apps/web/scripts`
